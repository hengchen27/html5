  1. Answer to question 1
    Inline styles:
    They are the styles that added as element attributes within an HTML document and thus apply to that element alone.
    Embedded styles:
    They are the styles added to the head of an HTML document.
    External styles:
    They are the styles that the website author creates and places within a CSS file and links to the page.

    I would use external styles to define a design for an entire website.
